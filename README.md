## JavaScript we see that everything is an object, in Go we have types. We can define our own types using the `type` keyword.

## Types

 - Floating ---> float32, float64
 - Complex ---> complex64, complex128
 - Array
 - Slice
 - Map
 - Struct
 - Pointer
 - Function
 - Interface
 - Channel
 - Error



## Package in Go
 - bufio


## Memory Management
 - new()
   - Allocate memory but no INIT
   - you will get a memory address
   - zeroed storage
 - make()
   - Allocate memory and INIT
   - you will get a memory address
   - non-zeroed
